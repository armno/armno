# Sinatra - A Ruby Framework

Sinatra is a lightweigt Ruby web framework.

## Installing Sinatra

### Install Ruby

```sh
$ sudo apt-get install ruby1.9.3
```

Then use `RubyGems` to install `sinatra` gem. (Ruby 1.9.3 shipped with RubyGems.)

```sh
$ gem install sinatra
```
